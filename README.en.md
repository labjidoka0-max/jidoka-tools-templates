## What's in here

A salesperson (no coding background) used Claude Code to cut **12 hours/month** of manual spreadsheet busywork. This repository shares the actual prompts and code.

**Non-engineers welcome.** No VBA, no Python expertise required.

## Quick start (30 seconds)

1. Clone this repo (or download ZIP)
2. Pick a prompt from `/prompts/en`
3. Paste it into Claude Code
4. Copy the returned code into Excel or Python

Done. No more busywork.

## Why trust this?

I'm not an engineer. I'm a salesperson who was tired of rebuilding the same Excel aggregations every week.

When Claude Code came out, I tried describing what I wanted in plain Japanese — and it gave me working code. That changed my workflow.

This repo is my working log. Every prompt here I actually use. Every piece of code was generated by Claude Code from the prompts shown.
