# excel-transcribe-ja — Excel転記プロンプト（入門サンプル）

## 用途
既存のテンプレExcelの決まった位置に、別のデータを貼り付けて「見出しはそのまま、中身だけ差し替える」パターン。

## 使い方
Claude Code を起動したディレクトリに `template.xlsx`（雛形）と `sales.csv`（差し込むデータ）を置いて、下のプロンプトをそのまま貼り付ける。

## プロンプト（丸コピOK）

```
template.xlsx の2枚目のシートに、sales.csv の内容を
「日付はA列、得意先名はB列、金額はC列」の対応で貼り付けてください。
3行目から下に入れてください。1行目と2行目の見出しはそのまま残してください。
出力は report_先週の日付.xlsx としてください。
```

## 出力イメージ
- `report_2026-04-14.xlsx` のような週次ファイル
- 1〜2行目の見出しは無傷
- 3行目以降に転記データ

## よくある失敗と修正
- 数式セルが値で上書きされる → 「数式が入っているセルには書き込まないでください」を追加
- 日付形式がバラつく → 「日付は YYYY-MM-DD で統一してください」を追加

## ポイント
- テンプレに貼り付けるタイプの仕事は、位置を「何行目のどの列」と座標で指示するのがコツ
- ファイル名に日付を入れると、毎週同じプロンプトを使い回せる
