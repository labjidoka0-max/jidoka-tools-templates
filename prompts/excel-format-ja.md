# excel-format-ja — Excel整形プロンプト（入門サンプル）

## 用途
バラバラな書式のデータ（電話番号、日付、金額など）を「決まった形」に揃える。

## 使い方
Claude Code を起動したディレクトリに `data.xlsx` を置いて、下のプロンプトをそのまま貼り付ける。

## プロンプト（丸コピOK）

```
data.xlsx のA列にある電話番号を、ハイフン区切りに揃えてください。
市外局番が抜けているものは「不明」と書いてください。
結果を同じファイルのB列に入れて、元のA列は消さないでください。
```

## 出力イメージ
- A列: 元データ（無傷）
- B列: 整形後（`03-1234-5678` の形、判定不能は「不明」）

## よくある失敗と修正
- 携帯番号と固定電話の区別がつかない → 「090/080/070で始まるものは携帯として扱ってください」を追加
- 全角数字が混ざっている → 「数字は半角に統一してから整形してください」を追加

## ポイント
- 整形系は「こうなっていたらどうする」という例外ケースを1つ添えると精度が上がる
- 元データを残す「同じ列に上書きしない」指定は毎回入れる
