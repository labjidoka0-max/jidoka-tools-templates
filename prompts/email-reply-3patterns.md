# メール返信 3パターン自動生成プロンプト

## 用途
受信メール1通に対して、返信の下書きを3パターン生成。
トーン（丁寧/標準/カジュアル）や結論（受諾/保留/辞退）を選んで貼るだけ。

1通12分→2分に短縮。

## Claude Codeへの指示（コピペ用）

```
以下のメールに対して、返信の下書きを3パターン作ってください。

【パターン構成】
- パターンA: 受諾（丁寧トーン）
- パターンB: 保留＋追加質問（標準トーン）
- パターンC: 辞退（やわらかく）

【出力形式】
各パターンに件名・本文・想定相手への注意点

【受信メール】
（↓ここに受信メール本文を貼る）
```

## 返ってくる出力の例

```markdown
## パターンA: 受諾（丁寧）

件名: Re: 打ち合わせの件について
本文:
○○様

お世話になっております。ジドウカLabの△△です。
ご提案いただいた件、ありがたくお受けいたします。
...

注意点: 返信タイミングは翌営業日以内推奨

## パターンB: 保留＋追加質問
（略）

## パターンC: 辞退（やわらかく）
（略）
```

## 応用

- パターン数を増やす: 「5パターン作って、温度感を柔らかい→硬いで並べて」
- 業界特化: 「SaaS営業の文脈で」と追加
- 英文対応: 「英語版も」と追加

## 動かないとき

返信の文脈が不足してる場合、Claude Codeが聞き返してきます。
「決定権は誰にありますか？」等の質問に答えれば精度UP。
