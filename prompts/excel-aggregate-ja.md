# excel-aggregate-ja — Excel集計プロンプト（入門サンプル）

## 用途
CSVを読み込んで、特定の列でグループ化して合計・並び替えし、Excelファイルに書き出す。

## 使い方
Claude Code を起動したディレクトリに `sales.csv` を置いて、下のプロンプトをそのまま貼り付ける。

## プロンプト（丸コピOK）

```
手元に sales.csv があります。列は日付・得意先名・商品名・数量・金額です。
得意先ごとに金額を合計して、合計が大きい順に並べた Excel ファイルを result.xlsx
として同じフォルダに出してください。元ファイルは変更しないでください。
```

## 出力イメージ
- `result.xlsx`（得意先名・合計金額の2列、降順ソート）
- 元の `sales.csv` は無傷

## よくある失敗と修正
- 金額列に「¥」や「,」が混ざっていて合計が0になる → プロンプト末尾に「金額は文字列でも数値化してから合計してください」を追加
- 得意先名の表記ゆれで別集計になる → 「大文字小文字と全角半角を揃えてから集計してください」を追加

## ポイント
- ファイル名・列名・出力先を全部日本語で明記する
- 「元ファイルを触らないで」と書くのは事故予防として毎回入れる
