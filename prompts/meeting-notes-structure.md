# 議事録 自動構造化プロンプト

## 用途
会議の雑な箇条書きメモを「決定事項 / 宿題 / 期限」の3列表に自動整理。
手作業で25分かかってた清書が30秒に。

## Claude Codeへの指示（コピペ用）

```
以下の会議メモを3列の表に整理してください。

【列構成】
- 決定事項: この会議で決まったこと（完了形で書く）
- 宿題（誰が何を）: 今後やるべきタスク、担当者付き
- 期限: 宿題の完了期限（日付か週単位）

【出力形式】
Markdown表 + 箇条書きの議論サマリ（3行以内）

【会議メモ】
（↓ここに会議のメモを貼る）
```

## 返ってくる出力の例

```markdown
## 議論サマリ
- 新プロジェクトのキックオフ決定
- 初期予算30万で合意
- 5月末の中間レビューが節目

## アクション

| 決定事項 | 宿題（担当） | 期限 |
|----------|-----------|------|
| 予算30万で実施 | 稟議書作成（田中） | 4/25 |
| プロジェクトマネージャーは山田 | 仕様書初版（山田） | 5/2 |
| 中間レビュー5月末 | 定例会議設定（鈴木） | 4/22 |
```

## 応用

- Slack への投稿フォーマットが欲しい → 「Slackで共有する投稿文も作って」と追加
- 英訳版も欲しい → 「上記を英語でも」と追加
- タスク管理ツール（Notion/Asana）にコピペしたい → 「Notionのテーブル形式で」

## 動かない場合

会議メモが短すぎる・長すぎる場合は Claude Code が聞き返してくれます。
「もっと詳しく」「もっと短く」で調整可能。
