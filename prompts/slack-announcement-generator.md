# Slack告知文 自動生成プロンプト

## 用途
長いメール文面や議事録から、Slack向けの短い告知文を自動生成。
読まれないテキストが、3行で見てもらえるSlack文面に。

## Claude Codeへの指示（コピペ用）

```
以下の内容をSlack告知用に整形してください。

【フォーマット】
- 冒頭に絵文字1つ（内容に合うもの）
- 1行目: 結論（誰に何を頼むか）
- 2行目: 背景or期限
- 3行目: アクション or リンク

【トーン】
- 敬体（です/ます調）だが親しみやすく
- 長すぎず短すぎず（合計3〜5行）
- @here などのメンションは付けない（私が自分で追加する）

【元の内容】
（↓ここに長文を貼る）
```

## 返ってくる出力例

```
:bulb: 新しい経費精算フローの試験運用について

来月から経費精算が新システムに移行します。4/30までにテスト参加お願いします。
詳細とマニュアルはこちら: https://...

質問があれば田中までどうぞ。
```

## バリエーション

- **ちょっとカジュアル版**: 「カジュアルに、🎉 使って」
- **重要告知版**: 「強調して、必読感出して」
- **ライト版**: 「1行で」

## 応用

元文面がメールの場合:
```
Slackで共有する用に要約してください。元はメールなので長いです。
（↓メール本文を貼る）
```

元文面が議事録の場合:
```
議事録の決定事項だけSlackで共有したいです。
（↓議事録を貼る）
```

## 時短効果

- 従来: 自分で要約10〜15分
- Claude Code: 30秒
- 月20件でも年間40時間浮く
