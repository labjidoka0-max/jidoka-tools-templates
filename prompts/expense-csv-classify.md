# 経費CSV 自動仕分けプロンプト

## 用途
経費精算で出力される摘要欄の文章を読んで、科目を自動分類。
30分の手作業→10秒に。

## Claude Codeへの指示（コピペ用）

```
経費CSVを読み込んで、摘要文から科目を自動分類するPythonスクリプトを書いてください。

【入力CSV】
- 列: 日付, 摘要, 金額
- 例: 2026-04-18, "JR新宿駅→東京駅", 320
- 例: 2026-04-18, "スターバックス コーヒー代（打合せ）", 1200

【分類ルール】
- 交通費: 「JR」「電車」「タクシー」「バス」「新幹線」「飛行機」を含む
- 会議費: 「コーヒー」「食事」「打合せ」「ミーティング」を含む
- 消耗品費: 「文房具」「ボールペン」「ノート」「USB」を含む
- 通信費: 「携帯」「インターネット」「電話」を含む
- 不明: 上記に該当しないもの

【出力CSV】
- 元の3列 + 「科目」列を追加
- 「不明」のものは最後に寄せる
```

## 応用

- ルール追加: 「会社の経費規定」を追加指示に入れる
- 明細の出処がSlackなら: 「Slack形式の発言ログから抽出する版」に変形
- 申請フォーム自動化: 「freee/マネーフォワード仕様のCSVに整形」

## 注意

経費データは個人情報。クラウドAIに渡すときは:
- 金額・日付は渡してOK
- 取引先の固有名詞は伏せる（「○○社」等）
- サンプルデータでテストしてから本番
